## üìò Course Overview
This course equips students with knowledge of the structure and graphic elements of web interface design. It focuses on two main phases:
1.  **Phase 1 (Theory & Survey):** Research methods, UI principles, and field surveys.
2.  **Phase 2 (Implementation):** Applying tools (Figma, Photoshop) and languages (HTML, CSS, Bootstrap) to build functional user interfaces.

### üéØ Learning Outcomes (CLOs)
By the end of this course, the projects in this repo demonstrate my ability to:
* Identify and apply UI design elements (colors, typography, icons) effectively.
* Proficiently use graphic design software (**Figma**, **Photoshop**) and web development tools (**VS Code**, **Bootstrap Studio**).
* Apply **HTML5**, **CSS3**, and **Bootstrap** framework to deploy responsive websites.
* Develop user flow, sitemaps, and wireframes for practical projects.
